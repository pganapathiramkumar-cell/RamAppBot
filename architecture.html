System Architecture
Client Layer
Web App Next.js 14 — dark-theme SPA deployed on Vercel
Mobile App Expo React Native 54 — iOS & Android via EAS
Gateway / Auth
API Gateway FastAPI — routes, rate limiting, JWT validation
Auth Service Supabase JWT — token issue & verification
Core Services
Document Service :8006 Upload, OCR, AI analysis pipeline (summary + entities + Mermaid)
Steer Service :8001 Strategic AI goals — CRUD, lifecycle, alignment scoring
Skill Service :8002 AI capability catalog — tagging, review, deploy lifecycle
AI Layer
Multi-Provider LLM Groq → NVIDIA NIM → Cerebras → Mock (auto-fallback on 429)
AI Orchestrator Parallel chain: summary + entity extraction + Mermaid generation
Data Layer
PostgreSQL Per-service databases via SQLAlchemy async + asyncpg
Redis Query-level cache (60 s TTL) with pattern invalidation on writes
ChromaDB Vector store for document embeddings (document-service)
Infra / Messaging
RabbitMQ Domain event bus — steer / skill / document events
Render Docker container hosting for all FastAPI services
Vercel Edge-deployed Next.js frontend (CDN + serverless)
Request Flow
Browser / Mobile → Vercel / CDN → API Gateway → JWT Verify → Microservice → Redis Cache → PostgreSQL → LLM Chain → RabbitMQ Event
